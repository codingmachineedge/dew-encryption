Dew Encryption

Windows and Linux file snapshots, VeraCrypt container quick-create workflows, per-container themes, open/close automations, and Git-backed history from Explorer, Nautilus, the CLI, or the GUI.

Windows install

Use the PowerShell bootstrap for dependency setup or download the release-built Inno Setup installer for GUI, CLI, icons, shortcuts, and Explorer actions.

Explorer first

Right-click a file or folder and choose dew encryption. Folders also include file-history start and manager actions. No Shift modifier is required.

History manager

Browse commits, inspect changed files, snapshot now, auto-watch changes, create archives, restore selected commits, and open the source or repo folder.

Container manager

Quick-create encrypted containers, store per-container theme colors and fonts, run multiple open/close hooks, and snapshot container files into dedicated Git history repositories.

Linux support

Install the same CLI and GUI with bash linux/install.sh. Nautilus scripts are added when available.
