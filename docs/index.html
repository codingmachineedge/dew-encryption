Dew Encryption

Windows Explorer right-click snapshots that commit selected files into a local Git repository and compress the repository with 7-Zip.

irm https://raw.githubusercontent.com/codingmachineedge/dew-encryption/main/installer/install.ps1 | iex

Automated install

The installer can bootstrap Python, Git, and 7-Zip with winget, clone the repo, install the package, and register the Explorer context menu under HKCU.

Explorer first

Right-click a file or folder and choose dew encryption. No Shift modifier is required.

GUI included

Run dew-encryption-gui to open the file manager and start the same snapshot workflow manually.
